| 指令                            | 動作       |
| ----------------------------- | -------- |
| reboot                        | 重新啟動     |
| shutdown -h now               | 關機       |
| /etc/init.d/[service] restart | 重啟服務     |
| ls /etc/init.d                | 查看現有服務   |
| ip                            | 查詢網路設定   |
| sudo su                       | 進入root模式 |
