#### 一般PCIe直通

```command
nano /etc/default/grub
```
將
```text
GRUB_CMDLINE_LINUX_DEFAULT="quiet"
```
改成
```text
GRUB_CMDLINE_LINUX_DEFAULT="quiet intel_iommu=on"
```
保存變更
```command
update-grub
```
重啟
```command
reboot
```

查看有無PCIe
```command
dmesg | grep -e DMAR -e IOMMU
```

上述完成 執行下面
```command
nano /etc/modules
```
新增
```text
vfio
vfio_iommu_type1
vfio_pci
vfio_virqfd
```
