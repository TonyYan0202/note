## 使用說明
* 新增/刪除 檔案時皆須追蹤&提交
* .gitignore 此檔案紀錄不需被追蹤的檔案

## 本地git

git版本查看
```
git -v
```

編輯者姓名與email
```
git config --global user.name <name>
git config --global user.email <email>
```

初始化
```
git init
```

庫狀態
```
git status
```

追蹤檔案
```
git add <filename/.>
```

提交檔案並說明
```
git commit -m <message>
```

查看歷史紀錄(簡化版)
```
git log <--oneline>
```

將當前還原點與指定還原點(point)的指定檔案(filename)比較
```
git diff <point> --<filename>
```

將指定檔案(filename)還原到指定還原點(point)的狀態 (需做新提交)
```
git checkout <point> --<filename>
```

將全數檔案還原到該點(point) 並刪除該點後所有紀錄(不可逆)
```
git reset --hard <point>
```

## 遠端多工

將此庫新增到遠端儲存庫(第一次上傳)
```
git remote add origin <git-url>
```

重新命名分支名稱(預設為master)
```
git branch -M <new branch name>
```

本地push到雲端 (-u 建立關聯[設定 git pull/status 的上游] origin 遠端名稱 main 本地名稱)
```
git push -u origin <branch name>
```

本地push到雲端 
```
git push
```

庫下載至本地
```
git clone <git-url>
```

更新本地庫
```
git pull
```

建立並切換至新分支
```
git checkout -b <new branch name>
```

## 其他

庫完整性
```
git fsck
```

庫連結
```
git remote -v
```
