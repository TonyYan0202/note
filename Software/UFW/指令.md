#### 啟用/停用

啟用
```
sudo ufw enable
```

停用
```
sudo ufw disable
```

#### 查看狀態

一般
```
sudo ufw status
```

詳細
```
sudo ufw status verbose
```

#### 規則設定

```
ufw <allow/deny> [-option]
```

| option  | command          |
| ------- | ---------------- |
| IP      | &lt;ip&gt;       |
| port    | 80 / 25565-25570 |
| service | ???              |
|         |                  |
|         |                  |

option: port: 80 / 25565-25580
ip

允許IP源
```
sudo ufw allow from <IP>
```

開放端口
```
sudo ufw allow 80
```

多個端口
```
sudo ufw allow 41901-41910
```
