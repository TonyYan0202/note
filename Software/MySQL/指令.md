#### 登出入
登入
```
mysql -u <帳號> -p
```

指定IP端口登入
```
mysql -u root -h 127.0.0.1 -P 3306 -p
```

登出
```
exit;
```

#### 內部指令

資料庫相關(DataBase)

| 指令                                                                             | 動作        |
| ------------------------------------------------------------------------------ | --------- |
| ```SHOW DATABASES;```                                                          | 查看所有資料庫   |
| ```CREATE DATABASE dbname;```                                                  | 建立資料庫     |
| ```CREATE DATABASE dbname CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;``` | 建立指定編碼資料庫 |
| ```USE dbname;c```                                                             | 切換資料庫     |
| ```DROP DATABASE dbname;```                                                    | 刪除資料庫     |

資料表相關(Table)

| 指令                                                                             | 動作           |
| ------------------------------------------------------------------------------ | ------------ |
| ```SHOW TABLES;```                                                             | 查看目前資料庫的所有表格 |
| ```DESCRIBE tablename;```<br>```SHOW COLUMNS FROM tablename;```<br>            | 查看資料表結構      |
| ```CREATE DATABASE dbname CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;``` | 建立資料表        |
| ```USE dbname;```                                                              | 刪除資料表        |
| ```DROP DATABASE dbname;```                                                    | 刪除資料庫        |
